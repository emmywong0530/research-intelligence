# Navigation Map

Research Intelligence uses persistent desktop left navigation across all pages.

## Primary Navigation

1. Home
2. Projects
3. Discovery
4. Library
5. Reading Hub
6. Ask Library
7. Synthesis
8. Research Gaps
9. Activity
10. Settings

## Project Tabs

Each project uses these tabs:

- Overview;
- Research Profile;
- Papers;
- Discovery;
- Synthesis;
- Gaps;
- Settings.

## Navigation Constraints

- Desktop is the primary design target.
- Data, text, vectors, and controls must stay within their panels.
- Long titles and metadata must wrap.
- Dense tables may use horizontal scrolling only when genuinely necessary.
- Display modes may change presentation but must keep information aligned.
